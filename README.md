# Análisis de Series Temporales - Apple (AAPL)

Proyecto de análisis financiero con Python para estudiar el comportamiento de las acciones de Apple Inc.

## Tecnologías utilizadas
- Python
- yfinance
- pandas
- matplotlib

## Objetivo
Análisis de precios históricos, retornos, volatilidad y tendencias.

## Cómo ejecutar
```bash
pip install yfinance pandas matplotlib
python apple_analysis.py
